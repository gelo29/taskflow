# TaskFlow

A Kanban-style task and project management app built as a learning project — progressing from static HTML/CSS through a modern React + Django REST Framework stack.

## Status

Phase 1: Foundations & tooling.

## Tech stack (planned)

- **Frontend**: HTML5, CSS3 (Flexbox & Grid), TypeScript, React
- **Backend**: Django + Django REST Framework
- **Database**: PostgreSQL
- **Deployment**: Docker, Railway / Fly.io

## Getting started

\`\`\`bash
git clone https://github.com/<your-username>/taskflow.git
cd taskflow
\`\`\`

Open \`src/index.html\` with the VS Code Live Server extension to preview.

\`\`\`
taskflow/

│ ├── styles/

\`\`\`

## Conventions

- \`camelCase\` for variables and functions
- \`PascalCase\` for components and classes
- \`kebab-case\` for filenames and CSS classes
- Conventional commits: \`feat:\`, \`fix:\`, \`refactor:\`, \`docs:\`, \`style:\`, \`test:\`, \`chore:\`
